# News Feed App

A Flutter News Feed App built with Clean Architecture.

## Features

- Get All Posts
- Get Single Post
- Create Post
- Update Post (PUT)
- Partial Update (PATCH)
- Delete Post
- Pull To Refresh
- Material 3 UI
- Error Handling
- Loading State
- Empty State

## Architecture

Clean Architecture

- Presentation
- Domain
- Data

## Packages

- flutter_riverpod
- dio
- retrofit
- auto_route
- json_serializable

## API

https://jsonplaceholder.typicode.com/posts

## Run

```bash
flutter pub get
dart run build_runner build --delete-conflicting-outputs
flutter run
```
